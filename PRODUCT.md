# Hermes Voice Product Context

register: product

Hermes Voice is a self-hosted voice console for an existing Hermes agent. The
primary user is a technically comfortable operator setting up and using a local
AI assistant with tools, memory, calendar access, file context, and voice I/O.

The interface should feel like a serious local instrument panel: modern,
legible, calm under pressure, and a little vintage without becoming decorative
or nostalgic. It should not read as a generic AI chat demo, SaaS landing page,
or neon sci-fi toy.

Design priorities:

- Fast comprehension of connection, mic, model, and tool state.
- Low cognitive load during live voice interaction.
- Professional settings controls for setup, model selection, updates, devices,
  and voice behavior.
- Warm dark neutrals, brass/amber operational accents, restrained cyan only for
  links or secondary status.
- Subtle physical cues: hairline borders, dense typography, signal meters,
  instrument-panel spacing.

Anti-references:

- Purple/blue AI gradients, oversized glass cards, bubbly chat-app styling,
  nested cards, novelty cyberpunk, and marketing hero composition.
